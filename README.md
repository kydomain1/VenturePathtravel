## ğŸš€ How to Use

### Opening the Website
1. Simply open `index.html` in any modern web browser
2. No server setup or installation required - it's a static website

### Navigation
- Use the top navigation menu to switch between pages
- Click on article cards to read full articles
- Use category buttons to filter content on homepage
- Use search bar to find articles (opens dedicated search page)

1. Open `index.html` in your browser
2. Type a keyword in the search bar (try "mountain", "coffee", or "minimalist")

5. Try different search terms or use the search bar on the results page
